如何将其他github仓库下载到本地后作为全新的仓库上传自己的remote？

```bash
git clone https://old-repo-url.git
cd repo

rm -rf .git

git init
git remote add origin https://github.com/YourUsername/new-repo.git
git add .
git commit -m "Initial commit"
git branch -M main
git push -u origin main
```
